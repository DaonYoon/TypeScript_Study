타입스크립트는 타입 정의가 필요하다.

JSDoc는 코멘트로 이루어진 문법이다. 함수바로위에 코멘트를 적어준다.
allowJs 는 js코드와 ts파일을 같이 사용하게 할수있다. (자바스크립트 코드를 수정하기 싫을떄 사용)


 npm i -D ts-node => 빌드없이 타입스크립트 실행됨. 프로덕션에서 쓰는 패키지는 아니고 개발환경에서 사용가능
 컴파일할 필요없이 타입스크립트 바로실행하게해줌
